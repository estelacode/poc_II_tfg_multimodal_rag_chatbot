# Nombre del Proyecto
Rediseñando la Experiencia de usuario con Inteligencia Artificial: Chatbot Especializado en la Industria de la Moda
Entregable parcial : PoC II: Chatbot multimodal
## Descripción

🟦 Pec 2: PoC : Chatbot Multimodal con LlamaIndex (Entregable Parcial)

UI: gradio
LLM: llama3.2-vision:11b and  llava:13b
Embeddings text: Cohere
Embeddings image: clip
Vector store: Chromadb
Datos: ficheros txt y jpg del directorio datasets_chatbots
Framework : LlamaIndex
